## Description longue

```
Kiddo transforme les corvées et responsabilités en véritable aventure pour tes enfants.

👨‍👩‍👧 POUR LES PARENTS
• Crée des tâches quotidiennes et ponctuelles
• Fixe des récompenses en pièces d'or
• Suis les progrès de chaque enfant en temps réel
• Protège chaque profil par un code PIN

⚔️ POUR LES ENFANTS
• Choisis ton avatar parmi 8 héros animés
• Gagne de l'XP et monte en niveau
• Débloque des classes : Guerrier, Mage, Archer…
• Échange tes pièces d'or contre de vraies récompenses

✅ COMMENT ÇA MARCHE
1. Le parent crée les tâches de la journée
2. L'enfant les valide une fois accomplies
3. Il gagne XP et pièces d'or
4. Les récompenses s'accumulent jusqu'au rachat

Kiddo rend l'autonomie amusante et la motivation durable.
```
